# HR-ANALYSIS-PROJECT
This project analyzes employee attrition patterns using HR data and presents insights through an interactive Power BI dashboard.
The goal is to understand employee risk rate, workforce distribution, and attrition trends across departments and gender.

The Power BI dashboard includes:

KPI Cards

Risk Rate (Attrition Rate)

Average Salary

Total Active Employees

Total Employees

Visualizations -

Pie Chart – Attrition by Gender

Bar Chart – Attrition by Department

Risk Rate Calculation

Tools & Technologies Used :

Python (Pandas) – Data cleaning and risk rate calculation

Power BI – Data visualization and dashboard creation

Excel/CSV – Data storage format

Insight :
Female employees aged 39 with a BBA qualification in Bangalore show a higher attrition rate (63%) compared to the overall average (48.71%), indicating a potential retention concern within this demographic segment.
